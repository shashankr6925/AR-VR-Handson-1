Hands-on 1: WebGL Static 3D Point Cloud 🌐✨
Description

This project sets up a WebGL environment to render a static 3D point cloud in the browser. Random 3D points are generated and displayed using vertex and fragment shaders.

Technologies

HTML5 🧾

JavaScript ⚡

WebGL 🎮

Outcome

Understand WebGL setup, shaders, and 3D point visualization.
